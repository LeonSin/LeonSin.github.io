Collections of Git Errors and Solutions

1.
problem description:
Git in Window 7,
$ git clone https://github.com/LeonSin/awesome-flask-todo
正克隆到 ‘awesome-flask-todo’…
fatal: unable to access ‘https://github.com/LeonSin/awesome-flask-todo/‘: error setting certificate verify locations:
CAfile: /bin/curl-ca-bundle.crt
CApath: none


Solution:
1) Fix the path to bin/curl-ca-bundle.crt,

git config —system http.sslcainfo “C:\Users\lenovo\AppData\Local\GitHub\PortableGit_7eaa994416ae7b397b2628033ac45f8ff6ac2010\bin\curl-ca-bundle.crt”

or

2)Choose to switch of SSL checks completely

git config —system http.sslverify false
