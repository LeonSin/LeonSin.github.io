Use vagrant or docker ?

Follow the discussion of this topic.
http://stackoverflow.com/questions/16647069/should-i-use-vagrant-or-docker-io-for-creating-an-isolated-environment

Read the answer from the author of docker and vagrant…

The short answer is that if you want to manage machines, you should use Vagrant. And if you want to build and run applications environments, you should use Docker.
